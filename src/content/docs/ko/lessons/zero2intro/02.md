## 서문

2장에서는 런치패드를 실제 사용 가능한 상태로 조정하는 과정을 다룹니다.
처음 개봉 후 컴퓨터에 연결할 때는 가능하면 도킹 스테이션/허브를 거치지 말고 직접 연결하세요.

## USB 드라이버 설치

:::tip[안내]
공식 사이트 접속이 안 되는 경우가 있습니다.
특히 일부 지역(예: 중국)에서는 국가/망 환경상 사이트의 핵심 리소스(Google Analytics, Facebook 등)에 접근하지 못해 접속이 실패할 수 있습니다.
이 경우 VPN 등 우회 접속으로 해결하거나, 미리 준비된 드라이버를 사용하세요: [다운로드](/ko/res/driver/)
:::

먼저 런치패드용 USB 드라이버를 설치합니다.
Novation 공식 사이트([https://novationmusic.com](https://novationmusic.com))에서 `Support - Download`로 이동한 뒤,
보유한 런치패드 세대를 선택합니다(MK1은 Legacy 메뉴).

![모델 선택](/images/tutorial/zerostart/02/choose-devices.jpg)

목록에서 `Novation USB Driver`가 USB 드라이버입니다.

- Windows 10 이상: 상단 최신 버전 설치
- Windows 10 미만: 하단 2.22 버전 설치

![드라이버 선택](/images/tutorial/zerostart/02/driver.png)

다운로드 후 런치패드를 연결하고 설치를 진행한 뒤, 완료되면 컴퓨터를 재부팅합니다.

---

### 설치 성공 여부 확인

시작 메뉴 검색에서 `장치 관리자`를 열고,
`사운드, 비디오 및 게임 컨트롤러`에서 `Launchpad + 모델명` 장치를 더블클릭해 속성을 확인합니다.

- 드라이버 버전이 2.22 초과인지
- 공급자가 Novation인지

![검색](/images/tutorial/zerostart/02/search.png)
![장치 관리자](/images/tutorial/zerostart/02/devicemanager.png)

정상이라면 드라이버가 올바르게 설치된 상태입니다.
정상이 아니면 다음을 수행하세요.

- `드라이버 업데이트` 클릭
- `드라이버 자동 검색` 실행
- 설치 완료 메시지 확인 후 재부팅

## 펌웨어 업데이트

런치패드를 사용하기 전에 펌웨어가 최신인지 확인해야 합니다.
구버전 펌웨어는 버그나 라이트 출력 불안정을 유발할 수 있습니다.

:::caution[주의]
업데이트 전에 반드시 아래를 확인하세요.

- USB 드라이버 설치 완료
- MIDI를 사용하는 소프트웨어 전부 종료(예: Ableton Live)
- 백신/보안 프로그램 종료
:::

### 런치패드 MK3 시리즈

MK3는 웹 기반 관리 방식(`Novation Components`)을 사용합니다.
웹페이지에서 커스텀 모드 추가, 펌웨어 업데이트 등 장치 관리 작업을 수행할 수 있습니다.
독립 실행 버전도 있지만 사실상 Electron 래퍼라 용량/점유 면에서 웹 사용이 더 효율적입니다.

#### 어떻게 업데이트하나요?

1. [Novation Components](https://components.novationmusic.com/) 접속
2. Chrome 또는 Edge 사용(예: Safari는 MIDI 장치 접근 불가)
3. 첫 접속 시 `Yes, I Agree` 선택

![쿠키/약관](/images/tutorial/components-cookies.png)

브라우저가 MIDI 장치 접근 권한을 묻는 경우 반드시 허용해야 합니다.
허용하지 않으면 페이지에서 런치패드를 감지하지 못합니다.

![MIDI 권한](/images/tutorial/chrome-allow-midi.png)

장치를 연결하면 자동 인식되며, 펌웨어가 낮으면 업데이트 알림이 표시됩니다.
`Update Firmware`를 눌러 업데이트를 진행합니다.

![장치 상태](/images/tutorial/components-states.png)

업데이트 도중 특정 버튼 입력을 요구할 수 있습니다.
화면에 표시된 버튼을 누르면 업그레이드가 시작되며, 예를 들어 런치패드 X는 `Capture MIDI` 버튼을 누르는 방식입니다.
완료되면 런치패드가 자동 재부팅됩니다.

![업데이트 확인](/images/tutorial/components-update-agree.png)

이로써 업데이트가 완료됩니다.

---

#### Components에 접속할 수 없다면?

Components는 해외 서버를 사용하므로 일부 환경에서 속도가 매우 느리거나 접속이 실패할 수 있습니다.
이 경우 수동 업데이트를 시도할 수 있습니다.

:::caution[주의]
아래 절차를 충분히 이해한 뒤 진행하세요.
잘못된 파일/절차는 펌웨어 손상을 유발할 수 있습니다.
:::

먼저 장치에 맞는 펌웨어를 다운로드합니다: [다운로드](/ko/res/fwupdater/#mk3-%EC%88%98%EB%8F%99-%ED%8E%8C%EC%9B%A8%EC%96%B4-%EC%97%85%EB%8D%B0%EC%9D%B4%ED%8A%B8)

**반드시 본인 장치 모델에 맞는 파일을 사용하세요.**

다운로드 후 [런치패드 펌웨어 도우미](https://lfu.arkitosekai.net)에 접속합니다.
커스텀 펌웨어 파일 기능은 고급 기능으로 기본 숨김 상태이므로, 페이지 로딩 후 키보드로 아래 순서를 입력합니다.

`↑，↑，↓，↓，←，→，←，→，`

그러면 드롭다운에 `커스텀 SysEx 파일` 항목이 표시됩니다.

![SysEx 항목](/images/tutorial/lfu-sysex.png)

방금 받은 펌웨어 파일을 선택하고 확인합니다.

![파일 선택](/images/tutorial/lfu-choose.png)

이후 페이지 안내에 따라 진행하면 런치패드가 자동 재부팅됩니다.

![BL 단계](/images/tutorial/lfu-bl.png)

이로써 펌웨어 업데이트가 완료됩니다.
BL 화면에서 멈춘 것처럼 보이면 케이블을 뽑았다가 다시 연결하세요. 즉시 벽돌 상태가 되는 상황은 대부분 아닙니다.

---

### 런치패드 MK2 / MK1 시리즈

MK2/MK1은 구형 방식으로, 펌웨어마다 전용 업데이트 프로그램을 개별 다운로드해 설치해야 합니다.

- 공식 다운로드: [Novation](https://downloads.novationmusic.com/novation)
- 공식 사이트 접속 불가 시: [다운로드](/ko/res/fwupdater/#launchpad-mk2-%EC%8B%9C%EB%A6%AC%EC%A6%88)

다운로드 후 런치패드를 연결하고 업데이터를 실행해 완료를 기다립니다.

![업데이터](/images/tutorial/lppromk1-updater.png)
![업데이트 진행](/images/tutorial/lppromk1-updater-progress.png)

이로써 업데이트가 완료됩니다.

## 런치패드 설정 조정

:::tip[안내]
이 항목은 Pro/X처럼 압력 감지 모델에만 필요합니다.
압력 감지가 없는 모델은 건너뛰세요.
:::

### 압력 감지(velocity)와 애프터터치 끄기

Pro/X 계열은 압력 감지와 애프터터치를 끄는 것이 일반적입니다.
여기서 끄는 것은 트리거 자체가 아니라, 패드를 두드리는 세기(velocity)와 애프터터치 값을 전달하는 기능입니다. 압력 감지를 켜두면 단순히 눌림/떼짐만 전달되는 것이 아니라 타격 강도까지 전달되며, 그 값이 음량 등에 반영되어 연주 시 소리가 들쭉날쭉해지기 쉽습니다. 라이트 프로젝트 연주에서는 보통 필요하지 않고, 주로 악기 음원 연주를 위한 기능이므로 끄는 편이 일반적입니다.

- 런치패드 X / 런치패드 프로 MK3
  - Setup 뷰 진입
    - 런치패드 프로 MK3: 측면 `Setup` 버튼 길게 누름
    - X: 상단 `Session` 버튼 길게 누름
  - 우측 사이드 키 2번째를 눌러 Velocity 뷰로 이동
  - 초록 버튼을 눌러 빨강으로 바뀌면 압력 감지 비활성화

![Velocity](/images/tutorial/zerostart/02/setup-vel.png)

  - 우측 사이드 키 3번째를 눌러 Aftertouch 뷰 이동
  - 빨간 버튼과 그 아래 버튼을 눌러 애프터터치 비활성화

![Aftertouch](/images/tutorial/zerostart/02/setup-aft.png)

- 런치패드 프로 MK1
  - Setup 진입
  - 3열 4번째/6번째 버튼 입력
  - 4열 6번째 버튼 입력
  - 압력 감지와 애프터터치 비활성화

![Pro MK1 설정](/images/tutorial/zerostart/02/setup-vel-promk1.png)

## Ableton Live 설치

Ableton Live는 Ableton이 개발한 DAW이며, 런치패드 사용의 필수 호스트 소프트웨어입니다.

- 설치 파일: [Ableton Live 받기](/ko/res/software/#ableton-live)

라이선스/활성화와 관련된 상세 절차는 본 문서에서 다루지 않습니다.
여러분도 아시다시피 여러 사정상, 이 문서에서는 "학습판"(비공식) 소프트웨어의 설치/활성화는 다루지 않으니 해당 부분은 각자 알아보시기 바랍니다.
