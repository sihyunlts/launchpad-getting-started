USB 드라이버는 Windows에서 런치패드를 사용하는 데 필수입니다. 드라이버를 설치하지 않으면 성가신 라이트 멈춤 현상이 자주 발생하고, 서로 다른 두 소프트웨어에서 동시에 런치패드를 사용할 수 없으며, Apollo Studio도 사용할 수 없습니다. 설치하면 여러 문제가 해결됩니다.

<details>
<summary>USB 연결 관련 자주 발생하는 문제 해결</summary>

Q: 컴퓨터에 런치패드를 꽂거나 뽑을 때 블루스크린이 뜹니다. 어떻게 해야 하나요?  
A: USB 드라이버가 원인일 수 있습니다. USB 드라이버가 2.22 이상의 버전으로 업데이트되어 있는지 확인하세요. 2.22 미만 버전은 블루스크린을 유발합니다. 케이블이 손상되어도 블루스크린이 발생할 수 있습니다.

Q: 컴퓨터가 런치패드를 인식하지 않고, 패드가 계속 불꽃놀이 표시만 합니다. 어떻게 해야 하나요?  
A: 이 문제는 여러 원인이 있을 수 있습니다.

- 첫째: 케이블에 문제가 있거나, 케이블의 저항(임피던스)이 너무 클 수 있습니다. 다른 브랜드 케이블로 바꿔 보세요. (MK3 시리즈를 제외하면) 기본 제공 케이블은 사용하지 않는 것을 권합니다.
- 둘째: USB 허브(도킹 스테이션)는 가급적 사용하지 마세요. 데스크톱이라면 케이스 전면 패널이 아니라 메인보드 후면 USB 포트를 사용하세요. USB 포트 자체가 고장났을 가능성도 있습니다.
- 셋째: 운영체제 전원 관리에서 고성능 모드를 켜 보세요. 브랜드 노트북이라면 해당 노트북 제어 센터에서 최고 성능 모드를 켜세요.

</details>
