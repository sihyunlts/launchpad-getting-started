title: "Max For Live 플러그인"
description: "여기에서 Live에서 사용하는 플러그인을 다운로드합니다."
---

amxd 플러그인(정식 명칭: Ableton Max Device)은 Max For Live를 기반으로 합니다.

이는 런치패드를 입문할 때 매우 중요한 부분입니다. Ableton Live는 원래 런치패드를 위해 특별히 설계된 소프트웨어가 아니기 때문에, 현재 대부분의 프로젝트가 amxd 플러그인을 사용해 런치패드 프로젝트를 더 잘 즐길 수 있도록 돕습니다.

:::tip[안내]
1. Lightweight, Iris 등의 플러그인이 필요하다면 Outbreak 카테고리를 선택하세요.
2. 플러그인을 찾을 때는 `Ctrl+F`로 검색하는 것을 권합니다.
3. [여기에 수록되지 않은 플러그인이 있나요? 여기를 클릭하세요](https://box.arkitosekai.net/about.html#%E8%81%94%E7%B3%BB%E6%96%B9%E5%BC%8F)
:::

<details>
<summary>플러그인은 어디에 넣나요?</summary>

다운로드가 완료되면 아래 디렉터리에 넣어 주세요.

Windows: `C:\\Users\\(사용자 이름)\\Documents\\Ableton\\User Library\\Presets\\MIDI Effects\\Max MIDI Effect`  
macOS: `/Users/(사용자 이름)/Music/Ableton/User Library/Presets/MIDI Effects/Max MIDI Effect`

</details>

## Plugin by mat1jaczyyy/Launchpad Lights

### Outbreak

![img](/images/amxd/outbreak_logo-bg.webp)

외부 네트워크에 접속할 수 있다면 원본 사이트에서 다운로드해 저자를 지원해 주세요: [방문](https://github.com/mat1jaczyyy/Outbreak)

---

Outbreak는 아래 플러그인을 모두 포함하는 플러그인 팩입니다.

- Lightweight: 조명 출력 플러그인
- Lightweight MIDIext Loader: 수동 로드 조명 변환 플러그인(수동으로 로드한 조명을 자동 로드로 변환할 수 있음)
- Iris: 간단한 조명 제작 플러그인(Resonator와 기능이 같지만 더 단순함)
- Depths Mixer/Depths Selector: 조명 레이어링 플러그인(여러 조명을 상하로 레이어링해 하나로 합치되 서로 간섭하지 않게 할 수 있음)
- Flip: 조명 반전 플러그인
- Delay: 조명 지연 트리거 플러그인
- Infinity: 조명 잔광(테일) 지연 플러그인
- Multi/Multi Reset: 원클릭 다중 효과 플러그인(Reset 리셋 모듈 포함, Multi Effect와 호환)
- Twist: 자동 페이지 전환 플러그인

이 통합 팩을 다운로드할 수도 있고, 개별 다운로드도 가능합니다.
