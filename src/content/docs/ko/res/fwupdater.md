## Launchpad MK3 시리즈

MK3 시리즈는 Novation Components 소프트웨어로 장치를 관리합니다. Components에서 업데이트하세요.

[Novation Components로 이동](https://components.novationmusic.com)

:::caution[주의]
브라우저가 WebMIDI 프로토콜을 지원하지 않으면 Novation Components를 사용할 수 없습니다.  
브라우저 호환 표는 아래와 같습니다.
:::

| 브라우저 | 플랫폼 | 호환 여부 |
| --- | --- | --- |
| Chrome | Windows & macOS | 호환 |
| Edge | Windows & macOS | 호환 |
| Opera | Windows & macOS | 호환 |
| Safari | macOS | 비호환 |
| Firefox | Windows & macOS | 버전 108 이상 필요 |
| 360 브라우저(고속 버전 포함) | Windows & macOS | 호환 |
| QQ 브라우저 | Windows | 호환 |

### MK3 수동 펌웨어 업데이트

공식 방식으로 펌웨어를 업데이트할 수 없다면, Components에서 모든 펌웨어를 추출해 두었습니다. 본인 모델에 해당하는 펌웨어를 다운로드한 뒤, 런치패드 펌웨어 도우미의 `사용자 지정 SysEx 파일` 플래시 기능으로 장치에 새 펌웨어를 올릴 수 있습니다. [사용 방법은 여기에서 확인할 수 있습니다.](/ko/lessons/zero2intro/02/#components%EC%97%90-%EC%A0%91%EC%86%8D%ED%95%A0-%EC%88%98-%EC%97%86%EB%8B%A4%EB%A9%B4)

:::note[권장]
Components에 전혀 접속할 수 없을 때에만 수동 업데이트 방식을 사용하기를 권합니다.
:::

- [Launchpad MINI MK3 최신 2.0.1 펌웨어](https://da-1302821495.cos.ap-chengdu.myqcloud.com/lpdupdater/lpminimk3/LPMiniMK3-467.syx)
