## Ableton Live (Win / Mac)

![img](/images/software/live_suite.ico)

Live는 Ableton에서 개발한 디지털 오디오 워크스테이션(DAW)입니다. Live는 음악 제작이나 라이브 공연을 위한 기능을 제공합니다. 런치패드는 Ableton과 협업을 통해 개발되었기 때문에, Live에 가장 많은 기능이 통합되어 있습니다.

Live는 기능과 가격에 따라 다양한 에디션으로 구분됩니다.

| 에디션 | 설명 |
| --- | --- |
| Lite | 런치패드를 구매하면 번들 소프트웨어로 제공되며, 기능이 매우 제한적입니다. |
| Intro | 구매 가능한 가장 저렴한 에디션입니다. 역시 기능이 매우 제한적입니다. |
| Standard | Intro에 비해 훨씬 많은 기능을 제공하지만, Max For Live가 포함되어 있지 않습니다. |
| **Suite** | **모든 기능을 사용할 수 있습니다.** |
| **Trial** | **Suite의 체험판입니다. 모든 기능을 사용할 수 있지만, 90일 후 프로젝트 저장 기능이 비활성화됩니다.** |

:::caution[주의]
- 대부분의 런치패드 프로젝트에서 사용되는 Max For Live가 Trial과 Suite에만 포함되어 있으므로, 런치패드 프로젝트를 원활하게 즐기려면 **Trial을 사용하거나 Suite를 구매해야 합니다.**
:::

:::tip[안내]
- Trial 버전은 90일 이후 저장 기능이 비활성화되기 때문에 프로젝트 제작에 사용하기는 어렵지만, **프로젝트를 연주하기 위한 목적으로는 무료로 제공되는 Trial도 충분하므로, Trial을 먼저 사용해 보는 것을 권장합니다.**
- 런치패드 구매 시 제공되는 Lite를 Ableton 계정에 등록하면, Suite로 업그레이드할 때 15% 할인된 가격으로 업그레이드가 가능합니다.
- Launchpad MK3 시리즈는 Live 10.1.13 이상 버전부터 사용 가능합니다.
:::

## Apollo Studio (Win / Mac)

![img](/images/software/apollostudioicon.webp)

Apollo Studio는 런치패드 조명 효과를 위해 만들어진 스탠드얼론 에디터로, Live와 Max For Live 없이도 화려한 조명 효과를 제작할 수 있습니다. Live와는 다른 방식으로 LED를 출력하기 때문에 런치패드의 64만 가지 색상을 모두 사용할 수 있습니다.

소리를 재생하는 기능은 지원하지 않기 때문에 Live와 함께 실행하여 사용해야 합니다.

### 다운로드

- [공식 홈페이지](https://apollo.mat1jaczyyy.com/)

:::tip[안내]
Windows의 경우 exe 확장자 파일을, macOS의 경우 pkg 확장자 파일을 다운로드해야 합니다.
:::

## UniPad (Android)

![img](/images/software/unipad.webp)

UniPad(유니패드)는 김지섭 및 유니패드 팀이 개발한 Android 전용 런치패드 시뮬레이터로, 실제 런치패드를 물리적으로 연결해 연주할 수 있습니다. 공식 설명에서는 런치패드를 기반으로 한 '리듬 게임(음악 게임)'으로 소개하고 있습니다.

:::caution[주의]
UniPad는 업데이트가 활발하지 않아 최신 안드로이드 버전에서 원활하게 작동하지 않을 수 있습니다. Android를 포함한 다양한 플랫폼에서 UniPack을 플레이하고 싶다면 [Amethyst Player](https://play.203.io)를 사용하는 것을 권장합니다.
:::

### 다운로드

- [공식 홈페이지](https://unipad.io)
