## Ableton Live

Ableton Live는 Ableton에서 만든 디지털 오디오 워크스테이션(DAW)입니다. Live로 음악을 자유롭게 창작하고, 라이브 공연과 즉흥 연주를 할 수 있습니다. Live는 런치패드와 깊게 통합되어 있습니다.

![img](/images/software/live_suite.ico)

Live에는 현재 다음 버전이 있습니다.

- Intro: 입문판
- Standard: 표준판
- Suite: 완전판
- Lite: 간소판(장치 번들, 단독 판매하지 않음)
- Trial: 체험판(Suite 30일 체험판, 기능 동일)

:::tip[안내]
- 하드웨어 번들 Lite를 Ableton 계정에 활성화하면, Suite로 15% 할인(85% 가격) 업그레이드가 가능합니다.
- Suite 버전에만 Max For Live 모듈이 포함되어 있으므로, 런치패드 프로젝트를 즐기려면 반드시 Suite 버전을 구매하세요.
- 체험판은 90일 동안 전체 내용을 무료로 체험할 수 있습니다. 기간이 끝난 뒤에도 계속 체험할 수 있지만 저장/다른 이름으로 저장이 불가능하므로, 프로젝트 제작에는 사용하지 않는 것이 좋습니다.
- Live 10.1.13 이상 버전만 Launchpad MK3 전 시리즈와 호환되며, 중국어 간체를 포함합니다.
- 충동 구매는 피하고, 먼저 체험판으로 경험해 보는 것도 문제없습니다 _(:з」∠)_
:::

<summary>Live의 최소 시스템 요구 사항(Live 12 버전)</summary>

### Windows

- Windows 10(1909 및 그 이상)
- Intel Core i5 프로세서 또는 AMD 멀티코어 프로세서
- 메모리 8GB
- 1366x768 디스플레이 해상도
- Link를 지원하는 ASIO 호환 오디오 하드웨어(고품질 오디오 성능 보장 가능)
- Live 라이선스 획득을 위한 네트워크 연결(추가 파일 다운로드 또는 Live 업데이트를 위해 고속 인터넷 권장)
- 기본 설치 시 시스템 디스크에 약 3GB 필요(디스크 여유 공간 8GB 권장)
- 추가 사운드 파일 저장을 위해 최대 76GB의 디스크 공간 필요(선택 사항)

### macOS

- macOS High Sierra 10.13 ~ Sonoma 14
- Intel Core i5 프로세서 또는 Apple Silicon
- 메모리 8GB
- 1280x800 디스플레이 해상도
- Core Audio 호환 오디오 인터페이스 권장
- Live 라이선스 획득을 위한 네트워크 연결(추가 파일 다운로드 또는 Live 업데이트를 위해 고속 인터넷 권장)
- 기본 설치 시 시스템 디스크에 약 3GB 필요(디스크 여유 공간 8GB 권장)
- 추가 사운드 파일 저장을 위해 최대 76GB의 디스크 공간 필요(선택 사항)
