## 서문

이 페이지에서는 런치패드를 사용하기 위한 초기 설정 방법을 다룹니다. 런치패드를 컴퓨터에 연결할 때는 안정적인 데이터 전송을 위해 가급적 USB 허브를 거치지 않고 직접 연결하는 것을 권장합니다.

## USB 드라이버 설치 (Windows)

:::note[참고]
macOS는 별도의 드라이버 설치가 필요하지 않습니다.
:::

런치패드의 원활한 인식을 위해 먼저 전용 USB 드라이버를 설치해야 합니다.

1. [Novation Downloads 페이지](downloads.novationmusic.com)에 접속합니다.
2. 현재 가지고 있는 런치패드를 선택합니다. (MK1 모델은 'Legacy' 페이지에서 확인할 수 있습니다.)
3. 해당 모델의 **Novation USB Driver**를 다운로드하여 실행합니다.

설치 완료 후 컴퓨터를 재시작하여 드라이버가 정상적으로 적용되도록 합니다.

### 설치 확인

1. 시스템의 **장치 관리자**를 실행합니다.
2. **'사운드, 비디오 및 게임 컨트롤러'** 항목을 확장합니다.
3. 연결된 'Launchpad [모델명]' 장치가 표시되는지 확인합니다.
4. 장치 속성에서 드라이버 공급자가 'Novation'이며, 버전이 2.22 이상인지 점검합니다.

장치가 정상적으로 인식되지 않는다면, 드라이버를 재설치하거나 다른 USB 포트에 연결하여 확인하시기 바랍니다.

## 펌웨어 업데이트

기기 사용 전 안정성 및 새로운 기능을 사용하기 위해 펌웨어를 최신 버전으로 업데이트해야 합니다.

:::caution[주의]
업데이트 진행 시 다음 사항을 꼭 확인해주세요.
- USB 드라이버를 먼저 설치해주세요.
- Ableton Live 등 런치패드를 제어하는 모든 소프트웨어를 종료해주세요.
- 펌웨어 업데이트 중에는 런치패드를 컴퓨터에서 분리하지 마세요.
:::

### 런치패드 MK3 시리즈 (X / Mini MK3 / Pro MK3)

MK3 시리즈는 웹 브라우저를 통해 펌웨어를 업데이트할 수 있습니다.

1. Chrome 또는 Edge 브라우저를 통해 [Novation Components](https://components.novationmusic.com/)에 접속합니다.
2. 브라우저의 MIDI 장치 접근 권한 요청 시 **허용**을 선택합니다.
3. 장치가 올바르게 연결되면 업데이트가 필요한 경우 **'Update Firmware'** 버튼이 표시됩니다.
4. 화면의 안내에 따라 업데이트를 진행하면 런치패드가 자동으로 재시작됩니다.

### 런치패드 MK2 / MK1 시리즈

구형 모델은 각 모델별 전용 업데이터 프로그램을 다운로드하여 실행해야 합니다.

1. [Novation 다운로드 페이지](https://downloads.novationmusic.com)에서 해당 모델의 전용 업데이터를 다운로드합니다.
2. 프로그램을 실행하고 화면의 안내에 따라 업데이트가 완료될 때까지 기다리면 런치패드가 자동으로 재시작됩니다.

## 장치 기본 설정 최적화

:::note[참고]
감압식 버튼이 탑재된 **Pro 및 X 모델** 사용자에게만 해당됩니다. 그 외 모델 사용자는 이 단계를 생략하셔도 무방합니다.
:::

### 감압 감지(Velocity) 및 애프터터치(Aftertouch) 비활성화

프로젝트 연주 시에는 일관된 소리 크기를 위해 감압 감지 기능을 비활성화하는 것이 일반적입니다.

**런치패드 X / 프로 MK3:**
1. 장치의 **Setup** 레이아웃으로 진입합니다.
   - **프로 MK3:** 측면 Setup 버튼을 길게 누릅니다.
   - **X:** 상단 Session 버튼을 길게 누릅니다.
2. 우측 버튼 중 **Velocity(VEL)** 페이지를 선택하여 기능을 비활성화합니다.
3. **Aftertouch(AFT)** 페이지를 선택하여 기능을 비활성화합니다.

**런치패드 프로 MK2:**
1. Setup 레이아웃에 진입합니다.
2. 3행의 4번 및 6번 버튼을 조작하여 설정을 구성합니다.
3. 4행의 6번 버튼을 사용하여 감압 기능을 비활성화할 수 있습니다.
