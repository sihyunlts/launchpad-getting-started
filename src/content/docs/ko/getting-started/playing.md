## 서문

본 시리즈의 마지막 장인 제4장에서는 실제 런치패드 프로젝트를 확보하여 실행하고, 시스템을 구성하여 퍼포먼스를 완성하는 과정을 다룹니다.

## 프로젝트 다운로드하기

런치패드 연주를 위한 프로젝트 파일은 주로 다음과 같은 경로를 통해 구할 수 있습니다.

- **YouTube:** 많은 런치패드 유튜브 채널에서 프로젝트 파일을 영상 설명란을 통해 배포합니다.
- **커뮤니티:** 런치패드 사용자들의 커뮤니티 사이트에서 자료를 찾을 수 있습니다.

대부분의 프로젝트는 Ableton Live에서 구동 가능한 프로젝트 형태로 제공되므로, 압축 파일 내부의 구성을 확인해야 합니다.

## 프로젝트 실행 및 파일 구조

다운로드한 압축 파일을 해제하면 일반적으로 다음과 같은 폴더 구조를 확인할 수 있습니다.

```text
📂 프로젝트 폴더/
├── 📂 Ableton Project Info/    # Ableton Live 프로젝트 고유 정보
├── 📂 Lights/                  # 라이트쇼용 MIDI 데이터 (.mid)
├── 📂 Presets/                 # 활용된 Max for Live 전용 장치 (.amxd)
├── 📂 Samples/                 # 사운드 소스 (.wav / .mp3 등)
└── 📄 프로젝트명.als            # 메인 실행 파일
```

`.als` 확장자 파일을 더블 클릭하여 Ableton Live에서 프로젝트를 실행해주세요.

---

## 미디어 파일 누락 해결 (Media Files Missing)

프로젝트 로드 시 하단 상태 표시줄에 주황색 경고창과 함께 **"Media files are missing"** 메시지가 나타날 수 있습니다. 이는 파일 경로의 변경이나 버전 차이로 인해 발생하며, 다음 순서에 따라 해결할 수 있습니다.

1. **자동 검색 (Automatic Search):** 주황색 경고창을 클릭하여 검색 패널을 연 후, 해당 프로젝트 폴더를 지정하여 누락된 파일을 자동으로 매칭시킵니다.
2. **수동 지정:** 자동 검색으로 해결되지 않는 경우, 좌측의 브라우저 패널의 'Current Project' 내부에서 직접 누락된 파일을 찾아 해당 누락 슬롯으로 드래그 앤 드롭합니다.

## 런치패드 모드 전환

원활한 연주와 라이트 연출을 위해 런치패드를 **User 모드** 또는 **Legacy 모드**로 전환해야 합니다.

| 모델 | 조작 방법 |
| :--- | :--- |
| **Pro MK3** | Setup 버튼 홀드 → 우측 하단 보라색 버튼 선택 (Legacy) |
| **X / Mini MK3** | 모드 버튼 홀드 → Setup 진입 → 우측 하단 보라색 버튼 선택 (Legacy) |
| **Pro MK1** | 상단 User 버튼 선택 (User 모드) |
| **MK2** | User1 버튼 선택 (User 모드) |

:::note[펌웨어 확인]
MK3 모델 사용 시 보라색 버튼이 표시되지 않는다면, 펌웨어를 최신 버전으로 업데이트해야 합니다.
:::

## 일반적인 프로젝트 설정

### 입력 및 출력 설정

대부분의 프로젝트는 사운드를 담당하는 **Audio 트랙**과 조명 효과를 담당하는 **Lights(MIDI) 트랙**으로 구분됩니다.

**Audio 트랙 설정:**
- **Input:** All Ins 또는 사용 중인 런치패드
- **Monitor:** Auto 또는 In
- **Record Arm:** 활성 상태 (빨간색)

**MIDI (Lights) 트랙 설정:**
- **Input:** All Ins 또는 사용 중인 런치패드
- **Output:** 사용 중인 런치패드
- **Output Channel:** 
    - 3세대(Pro MK3 / X / Mini MK3): **Ch. 1**
    - 2세대(Pro MK2 / MK2 / Mini MK2): **Ch. 6**
- **Monitor:** Auto 또는 In

### 페이지 전환 매핑

프로젝트의 페이지를 전환하기 위해 런치패드 버튼에 기능을 할당해야 합니다.

1. Ableton Live 우측 상단의 **MIDI** 버튼을 클릭합니다.
2. 프로젝트의 페이지를 제어하는 노브나 매크로 컨트롤을 선택합니다.
3. 런치패드 측면 버튼의 양쪽 끝을 동시에 눌러 범위가 올바르게 지정되도록 합니다.
4. 매핑 범위를 페이지 수에 맞춰 설정(예: 0-7)한 후 MIDI 버튼을 다시 클릭해 종료합니다.

## 특정 유형의 프로젝트 설정

### Project Manager 기반 프로젝트
Kaskobi가 주로 사용하는 'Project Manager' 플러그인이 포함된 경우입니다. 메인 트랙의 인터페이스에서 자신의 기기 모델을 선택하면 모든 트랙이 자동으로 설정됩니다.

### Apollo Studio 프로젝트
조명 효과를 위한 전용 소프트웨어인 **Apollo Studio**를 사용하는 경우입니다. 이 경우 Apollo Studio를 동시에 실행해야 하며, 필요한 경우 Live의 MIDI 신호가 Apollo로 전송되도록 추가 설정이 필요합니다.

### 여러 대의 런치패드를 사용하는 프로젝트
두 대 이상의 런치패드를 동시에 사용하는 경우입니다. 각 런치패드마다 독립적인 입력 및 출력 트랙을 설정해야 합니다.

## 연주의 시작

모든 설정이 완료되었다면 연주를 시작할 준비가 되었습니다.

- **튜토리얼 모드:** 많은 프로젝트가 연주 패턴을 익힐 수 있는 동영상이나 튜토리얼 MIDI 트랙을 포함하고 있으니 이를 참고하십시오.
- **반복 학습:** 런치패드 퍼포먼스는 높은 수준의 기억력을 요구합니다. 일정한 템포에서 충분히 연습하는 과정이 무엇보다 중요합니다.
