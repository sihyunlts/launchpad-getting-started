## 서문

**'처음부터 배우기'의 첫 번째 장에 오신 것을 환영합니다!** '처음부터 배우기'는 런치패드를 처음 접하는 입문자를 위해 작성된 가이드입니다.

이 페이지에서는 런치패드의 **작동 원리**를 중점적으로 설명합니다. 많은 강좌는 단순히 기본적인 설정을 적용하는 법만을 다루고 있기 때문에 그 이상으로 응용하기 어렵지만, 작동 원리를 파악하면 왜 그렇게 설정해야 하는지 이해할 수 있고, 나아가 스스로 런치패드를 설정하거나 프로젝트를 제작할 수 있게 됩니다. 조금 지루하더라도, 가장 중요한 페이지인 만큼 꼼꼼히 읽어주세요!

## 런치패드란 무엇이며, 어떻게 작동하나요?

런치패드(Launchpad)는 Novation사에서 개발한 MIDI 컨트롤러로, 음악 소프트웨어를 컨트롤하거나, 음악을 연주하기 위해 제작되었습니다. 런치패드는 악기처럼 다양한 소리를 연주할 수도 있고, 화려한 라이트쇼를 보여줄 수도 있지만, '컨트롤러'라는 이름에서 알 수 있듯이 **런치패드는 독자적으로 작동할 수 없습니다.** 컴퓨터와 **Ableton Live**를 비롯한 전용 소프트웨어가 있어야만 소리를 내거나 빛을 낼 수 있습니다. 소프트웨어 없이는 런치패드는 그저 벽돌에 불과합니다.

런치패드 버튼의 동작은 자유롭게 커스터마이징이 가능합니다. 모든 작동 방식은 정의하기 나름이며, 정해진 정답은 없습니다. [TAK의 K-POP Culture](https://youtu.be/zVPAKdSqNpY)처럼 여러 오디오 클립을 동시에 재생해 새로운 음악을 만들어낼 수도 있고, [lumin의 The Place](https://youtu.be/79GVykIxRDg)처럼 화려한 조명 효과를 만들어 음악에 맞는 비주얼라이저를 연출할 수도 있습니다.

## 런치패드를 즐기려면 어떤 컴퓨터가 필요한가요?

:::note[참고]
현재 런치패드의 사용 방식은 컴퓨터 기반과 모바일 기반으로 나뉩니다. 이 튜토리얼은 **컴퓨터 기반** 사용법만 다룹니다.

**컴퓨터:** 공식적으로 지원되는 일반적인 사용법입니다. 런치패드는 원래 컴퓨터 소프트웨어인 Ableton Live 전용으로 개발되었으며, 따라서 거의 모든 프로젝트가 Ableton Live로 제작됩니다.

**모바일:** UniPad나 Amethyst Player와 같은 비공식 앱을 사용합니다. 제조사의 공식 지원이 없으며, 최적화가 부족하기 때문에 런치패드의 모든 기능을 활용하기 어렵고, 프로젝트 구동 시 느리거나 지연이 발생할 수도 있습니다. 컴퓨터를 도저히 사용할 수 없는 상황이 아니라면, 컴퓨터를 사용하는 것을 강력히 권장합니다.
:::

런치패드를 구매하기 전, 컴퓨터 사양을 반드시 확인하세요. 특정 사양을 만족하지 못하면 소프트웨어를 원활하게 구동하기 어렵습니다. Ableton Live 12의 [공식 최소 사양](https://help.ableton.com/hc/en-us/articles/115001663530-Live-Minimum-System-Requirements)은 다음과 같습니다.

### Windows 최소 사양
| 항목 | 사양 |
| :--- | :--- |
| **운영체제** | Windows 10 (버전 22H2) 이상 |
| **CPU** | 5세대 Intel Core i5 또는 AMD Ryzen 이상 |
| **RAM** | 8 GB 이상 |

### Mac 최소 사양
| 항목 | 사양 |
| :--- | :--- |
| **운영체제** | macOS 11.7.10 Big Sur 이상 |
| **CPU** | Apple Silicon 프로세서 또는 Intel Core i5 이상 |
| **RAM** | 8 GB 이상 |

일반적으로는 macOS를 사용하는 것을 권장합니다. macOS는 MIDI 및 오디오 최적화가 잘 되어 있어 성능 면에서 유리합니다. 물론 Windows도 나쁘지 않지만, 추후 레이턴시(지연) 조정에 신경을 써야 할 수 있습니다.

최소 사양을 충족한다면 런치패드 입문을 고려해 볼 수 있습니다. 다만 프로젝트마다 컴퓨터에 요구되는 사양이 다를 수 있습니다. 대체적으로 복잡한 프로젝트일수록 요구 사양이 높아지며, 하나의 프로젝트가 8 GB - 32 GB 정도의 RAM을 사용하기도 합니다. 하지만 운영체제와 Ableton Live가 동적으로 자원을 조절해주므로 너무 걱정하지 않으셔도 되며, 최소 사양을 만족하면 대부분의 프로젝트는 원활하게 구동됩니다.

## 나에게 맞는 런치패드 모델 선택하기

런치패드는 크게 MK1, MK2, MK3 세 세대로 나뉘며, 소형(미니), 일반(X), 전문가용(프로) 모델로 구분됩니다. 여기서는 입문자에게 중요한 기능 위주로 간단히 정리합니다.

### MK3 시리즈 (최신 세대)
| 모델명 | 버튼 | LED | USB |
| :--- | :--- | :--- | :--- |
| **런치패드 프로 MK3** | 감압식 | RGB | USB-C |
| **런치패드 X (MK3)** | 감압식 | RGB | USB-C |
| **런치패드 미니 MK3** | 일반 | RGB | USB-C |

### MK2 시리즈 (단종됨, 중고로 구매 가능)
| 모델명 | 버튼 | LED | USB |
| :--- | :--- | :--- | :--- |
| **런치패드 프로 MK2** | 감압식 | RGB | USB-B |
| **런치패드 MK2** | 일반 | RGB | USB-B |
| **런치패드 미니 MK2** | 일반 | RG | Micro USB |

:::tip[구매 조언]
입문자라면 **런치패드 X**를 추천합니다. 최신 세대라 구매와 AS가 쉽고, 가격도 적당하며 라이트쇼에 필요한 모든 기능을 갖추고 있습니다. **런치패드 미니 MK3**는 버튼이 작아 연주하기에 불편할 수 있습니다.

전문적으로 연주하거나 작곡을 공부하고 싶다면 **런치패드 프로 MK3**도 좋지만, 공식 펌웨어의 결함으로 인해 라이트쇼 재생 시 끊김 현상이 발생할 수 있어 라이트쇼용으로는 적합하지 않을 수 있습니다.
:::

## 주요 용어 정리

| 용어 | 설명 |
| :--- | :--- |
| **Ableton Live<br>(에이블톤 라이브)** | 프로젝트 구동에 필수적인 소프트웨어입니다. |
| **UniPad<br>(유니패드)** | 안드로이드 환경의 런치패드 연주 앱입니다. |
| **Apollo Studio<br>(아폴로 스튜디오)** | 조명 효과 제작 및 재생을 위한 소프트웨어로, 2020년 이후 최신 프로젝트에서 자주 사용됩니다. |
| **Project<br>(프로젝트)** | 곡 연주에 필요한 파일 모음입니다. 에이블톤 라이브의 `.als` 파일과 오디오, 라이트 데이터를 포함합니다. |
| **UniPack<br>(유니팩)** | 유니패드에서 사용되는 프로젝트 형식입니다. 에이블톤 라이브와 호환되지 않는 별개의 파일 형식입니다. |
| **Lightshow<br>(라이트쇼)** | 실제 연주 없이 오직 조명 효과만 포함된 프로젝트입니다. |

## 추가 준비 사항

### 영어

이 튜토리얼은 한국어로 진행되지만, Ableton Live와 관련된 자료는 영어가 압도적으로 많기 때문에 기초적인 영어 실력이 있다면 훨씬 수월하게 배울 수 있습니다.

### 컴퓨터 활용 능력

앞으로의 튜토리얼에서는 기본적인 컴퓨터 조작법(압축 풀기, 단축키 사용, 파일 관리 등)을 알고 있다고 가정하고 진행합니다. 만약 컴퓨터 사용이 익숙하지 않다면 런치패드 사용에 어려움이 있을 수 있기 때문에, 컴퓨터의 기본적인 조작법을 먼저 익히는 것을 권장합니다.
